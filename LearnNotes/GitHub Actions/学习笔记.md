# GitHub Action 学习笔记

## GitHub Action 
- 在GitHub Repo 内自定且自动执行软体开发流程
- 建立或共享自动化工作
- 丰富的开发程序
- 达到自动化，持续整合与持续交付的目的
- 丰富的第三方套件可以使用

## Automation

- 自动使用Label 分类 Pull Request
